# Example Commands

## Object Editing

```text
Change the rider's jacket to red.
Turn the shirt blue.
Make the helmet black.
```

## Removal

```text
Remove the helmet from the person.
Remove the motorcycle from the background.
```

## Global Image Operations

```text
Blur the background.
Desaturate the image.
Apply a soft gaussian blur.
```

## Ambiguity Handling

```text
Change the left jacket to green.
Remove the person on the right.
Make the center object yellow.
```
